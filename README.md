# kukulkan-scripts


## Mostrar ayuda del comando

```sh
$ ./kukulkan.sh -h
```


## Descargar los proyectos de kukulkan

```sh
$ ./kukulkan.sh -init
```

## Actualizar todos los proyectos

```sh
$ ./kukulkan.sh -u
```


## Mostrar la rama en la que se encuentra cada proyecto

```sh
$ ./kukulkan.sh -b
```
